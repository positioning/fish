To try the package install R, start it and paste in the 
following red string: 

source('http://www.nielsensweb.org/trackit/install')

This will download and install the necessary package date 
and the trackit package itself. Current version is 0.2-6, 
and it was compiled 2011-08-26.   

If you decide to give this package a try, we would appreciate 
hearing about your experiences. Also we will assist you as 
much as we can. In other words please don't hesitate to send 
questions, suggestions, and comments to:
