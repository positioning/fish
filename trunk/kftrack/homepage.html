The KFtrack package

KFtrack is an add-on package for the statistical environment R to efficiently estimate movement parameters and predict 'most probable track' from raw geo-locations of tagged individuals.

Download

The KFtrack package for Linux kftrack_VERNR.tar.gz
The KFtrack package for Win* kftrack.zip
The reference manual pdf html

Install instructions

On Linux simply write: 
R CMD INSTALL kftrack_VERNR.tar.gz
On Win* use the menu 'Packages' and the sub-item 'Install package from local zip file'
