Electronic Tagging Geolocation Packages Repository

State space Kalman Filter
A suite of statistical models to estimate position errors, movement model parameters, and most probable positions from tracking data.

Visit our Google Code Project Page - this site has complete sets of source code and usage guides to help you utilize the geolocation packages for your work

Send comments or questions to: chihinl@usc.edu
How To Try
To run it fire up R and type (or paste) the following line: 
  source('http://www.nielsensweb.org/ukfsst/install')
This should be done in root/administrator-mode on Linux/Windows to allow each package to add to the R-documentation and library tree.
